Combining beauty and efficiency to display random posts


== Description ==

> What we have created is a combination of elegance and simplicity.
To be able to post random show site users. Simply install and activate the plugin Random post with ajax and only using a shortcode in any part of the site that you want to display random posts.

= Random Post with Ajax includes the following features: =
- Choice Categories
- Choice of Post Type
- Possibility to choose the number of posts
- Show Content (Excerpt)
- Display post date
- Show Thumbnail of post
- Beautiful design
- User friendly
- Responsive



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/random-ajax-post` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Ramdom Ajax Post' screen in WordPress
3.go to Random posts ->Random posts random post settings and set your settings
4. set do_action('wprndpst_settings') where you need to show.
5.for more information visit http://wpmen.ir/
